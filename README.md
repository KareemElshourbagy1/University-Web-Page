##  University Website
```
A responsive and modern university website built using HTML, CSS, and JavaScript.  
The website presents key information about the university,
 including programs, departments, contact details, and student resources,
 with a clean and professional design that adapts to all screen sizes.
```
 ```
> Features
- Fully responsive design compatible with all devices (desktop, tablet, mobile)
- Elegant landing page with smooth scrolling and navigation
- Sections for About, Courses, Campus, Contact, and more
- Interactive elements powered by JavaScript
- Organized and reusable CSS structure for easy customization
```
```
 > Tech Stack
- HTML5 – for structure and content  
- CSS3 – for design and layout  
- JavaScript (ES6) – for interactivity 
```
